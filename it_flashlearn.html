IT FlashLearn - Flashcard Learning Mobile App

Project Overview

System Name: IT FlashLearn

Aims: To enhance the study process for IT subjects by offering flip-to-reveal flashcards that promote active recall.

Technologies Used: Flutter, Firebase Firestore

Development Time: December 2024 (Assignment of Programming for Mobile Devices)

Features: -

  • Login and sign up functionality, with options for password reset via email.
  • Created flashcards for IT courses categorized by course and chapter with a flip-to-reveal feature.
  • Favorite flashcards feature, allowing users to save useful flashcards for easy access later.

(1) Sign Up & Login (Forgot Password)
First-time users of IT FlashLearn need to sign up before accessing the app. This demo showcases the sign-up process, followed by logging out and attempting to log back in. To demonstrate the "Forgot Password" feature, an incorrect password is intentionally entered. The system then sends a password reset link to the registered email, which the user clicks to reset their password successfully.
(2) Home Page
At the top of the home page, there is an animated IT FlashLearn logo, followed by a personalized greeting displaying the user's name and email. The page also features a carousel showcasing available courses and instructors. Below, student reviews of the system are displayed. This hardcoded page serves to showcase information without interactive features.
(3) Flashcards Page
The Flashcards page in IT FlashLearn organizes flashcards based on courses and their respective chapters. In the second tab, users will find a list of available courses. Upon selecting a course, the system displays all its chapters, allowing users to navigate to specific topics. Flashcards for each chapter can be viewed by clicking on them, with options to move through the cards using the next and previous buttons.
(4) Save Flashcard as Favorite
The Flashcards feature offers three options: Save as Favorite, Share as Text, and Share as Image. This demo highlights the Save as Favorite functionality. When users click the "Save as Favorite" button, the flashcard is added to their Favorites list, accessible in the third tab (represented by a heart icon). Users can also remove saved flashcards from this section.
(5) Share Flashcard (Text or Image)
Users can share flashcards as text or images through social media.
(6) Profile Setting
In the Profile Setting section of IT FlashLearn, users can modify their name, email address, and password. However, the current version of the app does not support changing the profile image. When a user clicks on the profile image, an alert dialog will display notifying them that the profile image cannot be changed at this time.

Problem Solved by IT FlashLearn

1. Time-Consuming Flashcard Creation

Existing platforms require students to manually create their own flashcards, adding unnecessary complexity and time to their study process. IT FlashLearn eliminates this by offering a library of ready-made flashcards, allowing students to focus on studying rather than creating content.

2. Disorganized Study Materials

Unlike the existing flashcard platform in the market, where users need to search separately for different chapters for a specific course, IT FlashLearn has categorized flashcards based on the course and its corresponding chapters. This method eliminates the need for students to search for different chapter notes for the same course.

Future Improvements

1. Expanded Course Selection

Currently, the app only offers flashcards for IT-related courses. A future update can incorporate more subjects across various fields, making it useful for a wider range of students.

2. Quiz Mode

In future, the IT FlashLearn can introduce a quiz mode, allowing users to test their knowledge based on the flashcards and reinforce learning through self-assessment.
