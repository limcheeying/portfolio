IT FlashLearn - Flashcard Learning Mobile App

Project Overview

System Name: IT FlashLearn

Aims: To enhance the study process for IT subjects by offering flip-to-reveal flashcards that promote active recall.

Technologies Used: Flutter, Firebase Firestore

Development Time: December 2024

Features: -

  • Login and sign up functionality, with options for password reset via email.
  • Pre-loaded flashcards for IT courses categorized by course and chapter with a flip-to-reveal feature.
  • Course-based flashcard categorization, making it easier to find flashcards for specific courses and chapters.
  • Favorite flashcards feature, allowing users to save useful flashcards for easy access later.

(1) Sign Up & Login (Forgot Password)
First-time users of IT FlashLearn need to sign up before accessing the app. This demo showcases the sign-up process, followed by logging out and attempting to log back in. To demonstrate the "Forgot Password" feature, an incorrect password is intentionally entered. The system then sends a password reset link to the registered email, which the user clicks to reset their password successfully.
(2) Home Page
At the top of the home page, there is an animated IT FlashLearn logo, followed by a personalized greeting displaying the user's name and email. The page also features a carousel showcasing available courses and instructors. Below, student reviews of the system are displayed. This hardcoded page serves to showcase information without interactive features.
(3) Flashcards Page
The admin dashboard offers an overview of system activities. It displays the active semesters and evaluation progress, along with several charts highlighting key metrics. All charts can be exported in SVG, PNG, or CSV format for further analysis and reporting. The final component is a calendar, allowing users to view the semester and evaluation durations.

In the Profile Setting section of IT FlashLearn, users can modify their name, email address, and password. However, the current version of the app does not support changing the profile image. When a user clicks on the profile image, an alert dialog will display notifying them that the profile image cannot be changed at this time.

Future Improvements

1. Expanded Course Selection

Currently, the app only offers flashcards for IT-related courses. A future update can incorporate more subjects across various fields, making it useful for a wider range of students.

2. Progress Tracking

In future, the IT FlashLearn can include a progress tracking feature, allowing users to monitor their learning progress.

3. Quiz Mode

In the future, the app can introduce an interactive quiz mode, allowing users to test their knowledge based on the flashcards and reinforce learning through self-assessment.
